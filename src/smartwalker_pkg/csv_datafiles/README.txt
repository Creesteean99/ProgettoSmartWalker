--------------[ITA]--------------

Le cartelle più rilevanti per il funzionamento del sistema sono 'odometry' e 'velocity'. Tutte le altre cartelle presenti in questa sezione sono state utilizzate esclusivamente per i nodi di test situati in 'src/Script Test'.
Il file 'custom' contiene i dati relativi all'ultima guida personalizzata creata. Ogni volta che viene generata una nuova guida, il contenuto di questo file viene sovrascritto. Una possibile implementazione futura prevede la creazione di una cartella dedicata per archiviare diverse guide personalizzate, permettendo all'utente di scegliere quale caricare all'interno di Rviz.

--------------[ENG]--------------

The most relevant folders for the system operation are 'odometry' and 'velocity'. All other folders in this section were used specifically for the test nodes located in 'src/Script Test'.
The 'custom' file contains data from the most recently created custom guide. Each time a new guide is generated, this file is overwritten. A potential future improvement involves creating a dedicated folder to store multiple custom guides, allowing the user to select which one to load within Rviz.
